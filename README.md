# ЛК — система управления лицевыми счетами

React-приложение + бэкенд + Telegram-бот для управления ЛК, статусами и запросами «поднять с отдыха».

## Деплой на Railway

1. Подключите репозиторий к Railway.
2. В настройках проекта добавьте переменную окружения:
   - **BOT_TOKEN** — токен Telegram-бота (получить у [@BotFather](https://t.me/BotFather)).
3. Railway соберёт проект (`npm run build`) и запустит сервер (`npm run start`).
4. Сервер раздаёт фронтенд и API; бот получает данные с сайта через API.

## Telegram-бот

После деплоя бот отвечает на команды:

- **/start** — приветствие и список команд.
- **/soon** — список ЛК «скоро на отдых» (оборот ≥ 4 млн).
- **/raise** — кого поднять с отдыха (активные запросы банкира).

Данные для бота синхронизируются с сайтом автоматически (при работе пользователей с таблицей).

## Локальная разработка

```bash
npm install
npm run dev
```

Фронтенд: http://localhost:3000 (Vite). API и бот при локальной разработке не запускаются (нужен отдельный запуск `npm run start` после `npm run build`).

## Запуск с ботом (как на Railway)

```bash
npm run build
npm run start
```

Подставь в переменные окружения свой `BOT_TOKEN` (или создай файл `.env` с `BOT_TOKEN=...`).

## Функционал

- Роли: пользователь, банкир, админ (админка: пользователи, логи, Telegram).
- ЛК: добавление, редактирование, удаление, оборот за квартал, вкладка «Скоро на отдых», вкладка «Поднять с отдыха» с уведомлениями банкира.
- Счётчики на вкладках: Список ЛК (N), Скоро на отдых (M), Поднять с отдыха (K).
- Бот в Telegram: команды /soon и /raise, связка с данными сайта.
